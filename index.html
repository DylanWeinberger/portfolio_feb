Web Development Portfolio

Jorts

Jorts is an app developed using Ruby on rails. It uses the geocoder gem and gmaps4rails. Users can create activities, see any activities within a 5 mile radius of the adress entered, and join those activites. There are alos features to chat with other users about those activities.

Web Development Portfolio

BowieBox

The Bowiebox is a jukebox built using Javascript and styled using SASS. It is fully responsive and the design changes drastically for mobile devices. There is also a seperate Bowie framework companion piece to this project.

Web Development Portfolio

FatCat Smuggling

FatCat Smuggling is a project built in Ruby on Rails used for a shipping mogul to keep track of their fleet. These tycoons can create vehicles and store them in their vehicle showroom. They can also assign smugglers to go on certain missions with these vehicles. They will also have a log of all these missions and the vehicles and smugglers associated with each.

Web Development Portfolio

Justice Junction

Justice Junction is a blog for intergalactic bounty hunters to post about missions with high credit rewards, and the newest techniques in carbonite freezing. The blog was built using Ruby on rails. We also used carrierwave for the images, illustrator for the design, and jquery for the animations.
